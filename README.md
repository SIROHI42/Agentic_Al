# Agentic AI Workflow Automation

**Project:** Agentic AI for Workflow Automation using LangGraph & LangChain  
**Tech Stack:** Python, LangGraph, LangChain, Google Generative AI API  

---

## Project Overview
Agentic AI automates goal-based workflows by orchestrating AI tasks.  
It leverages **LangGraph** for workflow management, **LangChain** for LLM pipelines, and **Google Generative AI API** for NLP tasks like text generation, summarization, and embeddings.

---

## Features
- Goal-based AI workflow execution  
- Google Generative AI API integration  
- Dynamic task orchestration with LangGraph  
- Language model pipelines via LangChain  
- CSV and structured data analysis  
- RAG (Retrieval-Augmented Generation) support
